Setting up FEniCS

Posted Mar 25 2021-03-25T15:44:12+00:00 by Homayoon

Installation

First of all you need to install Docker for Windows. Then you need to register an account on Dockerhub.

Getting FEniCS image

After you have done the steps above, you need to get legacy FEniCS image from its repository so run the command below in command prompt:
docker pull quay.io/fenicsproject/stable:latest
After getting the image, you need to run a container for the first time to instantiate it. Note that after running it for the first time, you can start the docker container already run before with the command below:
docker start [container name or container id or part of id]

Sharing folder

To share folder between host and container, for instance consider the scenario that you want to share the following path on Windows
D:\FEniCS
You have to open up Docker Dashboard->Settings->Resources->File Sharing
Then you need to add D:\FEniCS to allowed paths for sharing.
Now open up Windows Power Shell with administrator’s permission. cd to D:\FEniCS(Since we are using ${pwd} to share our current path and connect it to a path on container) then run the below commad
docker run -ti -v ${pwd}:/home/fenics/shared quay.io/fenicsproject/stable
This command will run a new container with the current working directory ${pwd} shared with the container at the path /home/fenics/shared from inside the container.
The syntax for the argument -v is /path/on/host:/path/in/container. Note that all paths are absolute paths.

Install ParaView

For importing and reading .vtu format. Install ParaView.

Using Matplotlib inside Docker container

We shoud run this command when starting the container

docker run -it -v ${pwd}:/home/fenics/shared -p 127.0.0.1:8000:8000 --name fenics --hostname fenics quay.io/fenicsproject/stable

Then we need to findout path of .matplotlibrc file if webagg gives us wrong address for output to view

After finding out the path to .matplotlibrc we edit the following line in .matplotlibrc file

1

webagg.address: 0.0.0.0

Instead of 0.0.0.0 we use our desired address, like localhost or 127.0.0.1

Using Jupyter Notebook

Basically all you have to do is make an instance of FEniCS docker image by running the following command

docker run --name notebook -w /home/fenics -v $(pwd):/home/fenics/shared -d -p 127.0.0.1:8888:8888 quay.io/fenicsproject/stable 'jupyter-notebook --ip=0.0.0.0'

Then you should go to docker dashboard and inspect the container created to copy the token. After copying the token all you have to do is go to the following address and paste the token and you are ready to use Jupyter notebook!

http://127.0.0.1:8888

import matplotlib
import matplotlib.pyplot as plt
plt.switch_backend('webAgg')
import numpy as np

# Data for plotting
t = np.arange(0.0, 2.0, 0.01)
s = 1 + np.sin(2 * np.pi * t*t)

fig, ax = plt.subplots()
ax.plot(t, s)

ax.set(xlabel='time (s)', ylabel='voltage (mV)',
       title='About as simple as it gets, folks')   
ax.grid()

fig.savefig("myExampleOfit.png")
plt.show()
