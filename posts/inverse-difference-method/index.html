Inverse Difference Method

Posted Jan 2 2021-01-02T09:59:37+00:00 by Homayoon

In mathematics, Thiele’s interpolation formula is a formula that defines a rational function \(f(x)\) from a finite set of inputs \(x_i\) and their function values \(f(x_i)\). The problem of generating a function whose graph passes through a given set of function values is called interpolation. This interpolation formula is named after the Danish mathematician Thorvald N. Thiele. It is expressed as a continued fraction, where ρ represents the reciprocal difference:

function res = funcPhi(f, x)
    n = length(x);
    if(n>1)
        if(n==2)
            res = (x(1)-x(2))/(f(1)-f(2));
            %if(isinf(res))
            %    res = inf;
            %end
        else
            % Suppose x = [0 1 2 3] then (x~=x(length(x)-1)) outputs [0 1
            % 3] (It omits the element before the last element)
            newx = x(x~=x(n-1));
            newf = f(f~=f(n-1));
            res = (x(n-1)-x(n))/(funcPhi(f(1:n-1),x(1:n-1))-funcPhi(newf,newx));
            %if(isinf(res))
            %    res = inf;
            %end
        end 
    end
end

clear;
clc;
x = [0 1 2 3];
f = [0 -1 -2/3 9];
%----------------
%x = linspace(-1,10,6);
%x = 7:-0.5:0.5;
%f = 1./x; % function f(x)=1/x
%----------------
n = length(x);
T = zeros(n,n);
T(1:n) = f;
for i=2:n
   for j=i:n
      T(j,i)=funcPhi([f(1:i-1) f(j)] ,[x(1:i-1) x(j)]);
   end
end
% Want to change x to symbol for representing equation
xcoords = x;
Answer = [xcoords' T];
%num2str(Answer, 12)
for I = 1:n
    %fprintf(OUP, '%11.8f ', XX(I+1));
    for J = 1:n+1
    fprintf('%11.8f     ', Answer(I,J));
    end
    fprintf('\n');
end
phiArray = diag(T);
syms('x');
P = 0;
for i=length(phiArray):-1:2
   P = (x-xcoords(i-1))/(P+phiArray(i)); % Solving continued fraction from bottom to top
end
P
%POINT = 3.4;
POINT = 2;
subs(P,x,POINT)% Substituting variable x with value at variable POINT
vpa(subs(P,x,POINT))
%fplot(P)
